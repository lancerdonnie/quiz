# Quiz App

A simple quiz app

### Technologies

- React/Redux
- Sass
- Typescript
- React-Spring
- Tailwind CSS

### How to run

- Clone this project
- Install nodeJS
- Run npm install in your terminal

#### Commands

- Start: Run npm start in your terminal
- Build: Run npm run build in your terminal
- Test: Run npm test in your terminal
